# docs/solutions/ — the craft ledger

One durable engineering or review lesson per file, captured at the moment it's
earned, organized by category with YAML frontmatter for search. This is the
compound-engineering convention: `/ce:compound` captures a solved problem, and
the `learnings-researcher` agent searches here before similar work begins, so
lessons get applied instead of re-learned.

Division of labor (one home per lesson, never two):
- concept George is learning → `~/cursor-course/COURSE-LOG.md` flashcards
- hard constraint agents must obey → `AGENTS.md`
- where the work stands → the delta doc's build-state cursor
- transferable craft → here
- review technique that recurs across weeks → `~/cursor-course/review-week/PLAYBOOK.md`
